### nanobox-hapi-mongo
A simple RESTful API built with [hapijs], [mongodb] and [nanobox]

#### Running Tests
The project uses [Code], [Sinon] and [Lab] for testing.

To run the tests:
```
$ yarn test
```

#### Start Server
To start the server:
```
$ yarn start
```
Visit [`localhost:11001/documentation`](http://localhost:11001/documentation) to play around.
